P-Do3 Generate human-readable documentation

Easy-to-use documentation is critical for the wide adoption of the vocabulary. As described in subsection 4.3.1, we have two different types of URIs. In case that during vocabulary creation slash URIs are used for identifying resources then tools like Schema.org documentation generation22 should be used for documentation generation. Tools like Parrot23 are appropriate if hash URIs or combination between slash and hash URIs are used for identifying resources.
