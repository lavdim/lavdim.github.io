P-V1 Syntax validation

When collaborating directly on the vocabulary source code, syntax checking is of paramount importance. Ideally syntax checking is directly integrated into the editor and committing violating code is not possible. For example, tools such as Parrot24 or Web-based tools such as: RDF Validation Service25 or OWL2 Validator26 can be used for finding common typos and syntax errors.
